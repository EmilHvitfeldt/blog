Day 3: details

Welcome back for the 3rd day of the #packagecalendar, today we will be talking about another little nifty package called details package by Jonathan Sidi.

the package is available from CRAN and can be downloaded with

install.packages("details")

Suppose you want to hide some information in your html document, while still allowing the user to see it if they want to. In plain html something like that would normally be done using the details tag.

<details>
 <summary>Secret list of names</summary>
 Aura
 Dona
 Danyelle
 Maribel
 Evonne
 Elly
 Joye
 Monet
 Donny
 Georgeanna
</details>

Which would result in something like this:

Secret list of names 


Aura
Dona
Danyelle
Maribel
Evonne
Elly
Joye
Monet
Donny
Georgeanna


Putting R objects or output into a details tag used to be quite frustrating. Presenting details! Now all you have to do is pass (piping works too) what you want hidden into details().

library(details)
library(magrittr)

secret <- c("Madeline", "Tom", "Shavon", "Krystyna", "Callie",
            "Damon", "Shellie", "Bebe", "Candis", "Laure")

details(secret)

Madeline
Tom
Shavon
Krystyna
Callie
Damon
Shellie
Bebe
Candis
Laure
